# Sequential Thinking MCP

**Идентификатор:** `user-sequential-thinking`  
**Статус:** подключён в проекте (доступен агенту)

## Назначение

Пошаговое структурированное рассуждение для сложных задач: разбиение на шаги, проверка допущений, планирование.

## Когда применять

- Архитектура проекта (сайт, лендинг, бот).
- Сложный рефакторинг или миграция стека.
- Планирование этапов разработки и приоритетов.
- Задачи с множеством условий и ограничений.

## Примечание

Не заменяет обычную разработку; полезен там, где нужна явная цепочка рассуждений и шагов.
