# Lighthouse MCP

**Статус:** рекомендуемый к подключению

## Назначение

Запуск аудита Lighthouse для страницы: Performance, Accessibility, Best Practices, SEO. Метрики Core Web Vitals (LCP, FID/INP, CLS) и рекомендации по улучшению.

## Когда применять

- Оптимизация лендингов и сайтов после вёрстки.
- Проверка перед релизом (производительность и доступность).
- Поиск причин низких оценок и узких мест.
- Соответствие целевым показателям (например, LCP < 2.5s).

## Подключение

- Через Cursor MCP (если есть готовый Lighthouse MCP server).
- Либо отдельный скрипт/CI (npm run lighthouse) с сохранением отчётов.
