# Telegram Bot MCP

**Статус:** рекомендуемый к подключению для разработки Telegram-ботов

## Назначение

Интеграция Telegram Bot API с Cursor: отправка и получение сообщений, управление ботом из агента. Удобно для тестирования сценариев и отладки без постоянного переключения в Telegram.

## Варианты

1. **telegram-bot-mcp-server** (TypeScript, NPX)  
   - Установка: `npx telegram-bot-mcp-server`.  
   - В настройках Cursor MCP указать Bot Token от @BotFather.  
   - Умная разбивка длинных сообщений (лимит 4096 символов).

2. **Telegram MCP (Python/Telethon)**  
   - Широкий набор инструментов (чаты, сообщения, группы).  
   - Требуются API ID и API Hash из my.telegram.org.

3. **Другие реализации**  
   - MsgMCP, gramjs-based серверы — выбор по стеку (Node/Python) и нужному набору операций.

## Подключение

- Cursor → MCP → добавить сервер (command + env с токеном или API credentials).
- Bot Token или API ID/Hash не коммитить в репозиторий; использовать переменные окружения.

## Когда применять

- Разработка и отладка Telegram-бота.
- Автоматические проверки сценариев (отправка команд, проверка ответов).
- Тестирование обработчиков и диалогов без ручного ввода в клиенте.
