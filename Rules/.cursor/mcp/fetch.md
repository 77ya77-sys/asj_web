# Fetch MCP

**Статус:** рекомендуемый к подключению

## Назначение

Получение содержимого по URL (HTTP GET). Проще, чем Firecrawl: сырой HTML или текст без полноценного парсинга.

## Когда применять

- Быстрая проверка, что страница отдаётся и какой контент там есть.
- Чтение API (если возвращает JSON/XML).
- Сценарии, где не нужен глубокий парсинг структуры страницы.
