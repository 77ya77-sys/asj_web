# Context7 MCP

**Идентификатор:** `user-context7`  
**Статус:** подключён в проекте (доступен агенту)

## Назначение

Получение актуальной документации и примеров кода для любых библиотек и фреймворков.

## Использование

- Документация и примеры: React, Next.js, Vue, Tailwind CSS, GSAP, Framer Motion и др.
- Telegram Bot API, grammY, node-telegram-bot-api — при разработке ботов.
- Любые npm-пакеты: поиск по имени, получение актуальных сниппетов.

## Инструкция сервера

Use this server to retrieve up-to-date documentation and code examples for any library.

## Когда применять

- Нужна актуальная документация по библиотеке.
- Нужны примеры кода по API или хукам.
- Разработка сайта, лендинга или бота с новым стеком.
