# Puppeteer MCP

**Идентификатор:** `user-puppeteer`  
**Статус:** подключён в проекте (доступен агенту)

## Назначение

Автоматизация браузера для тестирования и сценариев: навигация, клики, ввод текста, скриншоты, выполнение JS в странице.

## Основные инструменты

- Навигация по URL.
- Клики, ввод в поля, скролл.
- Скриншоты страницы или элементов.
- Выполнение кода в контексте страницы (evaluate).

## Когда применять

- E2E-тесты лендингов и сайтов.
- Проверка форм, навигации, ключевых сценариев.
- Снятие скриншотов для документации или регрессий.
- Автоматизация сценариев после деплоя.
