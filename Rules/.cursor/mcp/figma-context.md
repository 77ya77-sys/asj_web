# Figma Context MCP

**Статус:** рекомендуемый к подключению

## Назначение

Анализ макетов Figma и извлечение дизайн-токенов: цвета, типографика, отступы, компоненты. Ускоряет вёрстку по макету и соблюдение дизайн-системы.

## Подключение

- В настройках Cursor MCP добавить Figma Context server.
- Указать Figma API token (Personal Access Token в настройках Figma).
- При необходимости — file key или node ID макета.

## Когда применять

- Вёрстка лендинга или сайта по макету Figma.
- Построение дизайн-системы (токены, компоненты).
- Уточнение отступов, шрифтов, цветов без ручного замера.
