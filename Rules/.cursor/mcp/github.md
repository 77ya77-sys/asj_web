# GitHub MCP

**Статус:** рекомендуемый к подключению

## Назначение

Работа с репозиторием через MCP: просмотр файлов, коммитов, веток, pull requests. Помогает при онбординге, ревью и доработках по коду.

## Когда применять

- Анализ структуры репозитория и истории изменений.
- Поиск мест, связанных с задачей (по коммитам или PR).
- Понимание текущего состояния кода при доработках.
- Проверка конфликтов и состояния веток.

## Подключение

- Cursor MCP settings.
- GitHub Personal Access Token с правами repo (и при необходимости read:org).
