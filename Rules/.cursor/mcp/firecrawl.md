# Firecrawl MCP

**Статус:** рекомендуемый к подключению

## Назначение

Скачивание и парсинг веб-страниц по URL. Извлечение структуры, текста, метаданных. Подходит для анализа конкурентов и референсов.

## Подключение

- Cursor MCP settings.
- API key с firecrawl.dev (или аналог, если сервер другой).

## Когда применять

- Исследование сайтов конкурентов (структура, блоки, тексты).
- Сбор референсов для лендинга или раздела.
- Анализ существующего сайта перед редизайном.
- Извлечение контента для миграции или аудита.
