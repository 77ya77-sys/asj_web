## Этап 2: Правила проекта

- [x] Запретить начало кода до прохождения интервью и формирования документов

## Этап 3: Skills

- [x] **project-interview**: папка `.cursor/skills/project-interview`, файл `SKILL.md`
  - Задача: проведение интервью (тип проекта, цель, аудитория, регион, функции, монетизация, стек)
  - Результат: ответы для формирования project-spec
- [x] **generate-project-spec**: папка `.cursor/skills/generate-project-spec`, файл `SKILL.md`
  - Задача: преобразование ответов интервью в структурированный `project-spec.md`
  - Содержание: описание, цели, аудитория, функции, архитектура, стек
- [x] **generate-roadmap**: папка `.cursor/skills/generate-roadmap`, файл `SKILL.md`
  - Задача: создание `roadmap.md` с этапами разработки
  - Этапы: инициализация, структура, верстка, функционал, интеграции, оптимизация, тестирование
